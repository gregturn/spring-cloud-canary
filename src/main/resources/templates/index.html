Automated Canary Analysis

Automatically analyzing your deployed application against its predecessor is critical to the success of your customers.

With this service, you simply provide a link to your cluster, and it will assess if performance has degraded, giving you back a response in seconds.

  • Analyze 'sdrdemo-production-gturnquist'
  • Analyze 'sdrdemo-production-gturnquist' and just return the status

If you click on the hyperlink above, you'll get back a JSON document detailing a set of canary checks performed against an application cluster.

  • Itemized list of each check
  • Status result of each check
  • Descriptive message of each check
  • Summary status at the bottom to vote yeah/nay on pipeline deployments
